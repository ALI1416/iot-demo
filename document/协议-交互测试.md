# 协议-交互测试

## Interact3000000 设置通信地址

`$iot/request/123/0/3000000`

```json
{
  "requestSn": 108276097690370049,
  "request": {
    "uri": "mqtt://127.0.0.1:1883",
    "username": "account",
    "password": "pwd"
  }
}
```

`$iot/response/123/0/3000000`

```json
{
  "timestamp": 1712627499659,
  "requestSn": 108276097690370049,
  "errorCode": 0
}
```
